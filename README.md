# School Portal website with map pathfinding using Dijkstra's algorithm

This is a school management portal website where administrators can encode grades of students and manage the website data includes the home page, feedback list, student postings. This website also allows students to create a posting board where other students can interact with, they also can view their grades and status.  This website also includes a sample map and implemented pathfinding using Dijkstra's algorithm.
